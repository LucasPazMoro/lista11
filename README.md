# Lista 11
Projeto desenvolvido no curso DEV-TI (Unoesc / Fapesc)

## Como rodar?
Tenha o Node js instalado e rodo os seguites comandos
```
npm install
npm run dev
```

Após isso, basta abrir o link disponibilizado no terminal e o projeto estará rodando!

## Tecnologias
- HTML / CSS / javaScript
- Node.js
- Express
